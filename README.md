# 🛒 GroceryMate - Grocery List Manager

A full-stack web application to manage grocery shopping lists.

## 🔗 Live Links
- **Frontend:** https://grocery-mate-two.vercel.app
- **Backend API:** https://grocery-mate.onrender.com

## 🛠️ Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MySQL (Railway)
- Auth: JWT
- Deploy: Vercel + Render
